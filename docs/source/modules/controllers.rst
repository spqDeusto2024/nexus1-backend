Controllers Module
=================

The `controllers` module contains the business logic for handling different resources in the application, such as tenants, rooms, roles, and more. Each controller file is responsible for a specific resource or group of resources.

Files in the `controllers` module:
-----------------------------------

- `administrator_handler.py`: Handles requests related to administrators.
- `auth_handler.py`: Handles authentication-related requests.
- `dormitory_handler.py`: Handles requests for dormitory data.
- `parameter_handler.py`: Handles requests for parameter data.
- `parameter_room_handler.py`: Handles requests for parameter room data.
- `relationship_handler.py`: Handles relationships between resources.
- `role_handler.py`: Handles role-related requests.
- `room_handler.py`: Handles room-related requests.
- `shelter_handler.py`: Handles shelter-related requests.
- `tenant_handler.py`: Handles tenant-related requests.
- `tenant_relationship_handler.py`: Handles relationships between tenants and other resources.

Description:
------------
This module is responsible for mapping requests to functions that interact with the database and return the appropriate data to the user.
